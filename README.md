# App de Francês A1: Missões Interativas

Este projeto é um protótipo de uma aplicação educativa gamificada para o aprendizado de Francês (Nível A1). O objetivo é ensinar vocabulário e pronúncia através de missões baseadas em situações cotidianas.

## 🚀 Funcionalidades
- **Reconhecimento de Voz:** Prática de pronúncia em tempo real usando a Web Speech API.
- **Sistema de Gamificação:** Acumulação de pontos (XP) e progressão por níveis (desbloqueio de novas missões).
- **Interface Intuitiva:** Focada em dispositivos móveis, simulando uma experiência de jogo.

## 🎮 Como Funciona
1. **Missões:** O aluno deve completar desafios de áudio (ex: "Le réveil", "Bonjour") para avançar.
2. **Progressão:** O Nível 2 só é desbloqueado após a conclusão bem-sucedida do Nível 1.
3. **Feedback:** A aplicação fornece retorno imediato sobre a pronúncia do utilizador.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura da aplicação.
- **CSS3:** Design responsivo e animações (Gamificação).
- **JavaScript:** Lógica de jogo e integração com o reconhecimento de voz do navegador.
